## 📱 الاستجابة (Responsive)

### Desktop (> 768px)
```
- عرض كامل للـ Header
- البنر يعرض جميع العناصر جنباً إلى جنب
- نص "خروج" يظهر مع الأيقونة
- الإشعارات بعرض 384px
```

### Tablet (768px - 1024px)
```
- عرض مناسب للـ Header
- البنر قد ينتقل لصفين
- نص "خروج" يظهر
- الإشعارات بعرض كامل
```

### Mobile (< 768px)
```
- قائمة همبرغر للـ Sidebar
- البنر يعرض العناصر عمودياً
- نص "خروج" يختفي (أيقونة فقط)
- الإشعارات ملء الشاشة
```
